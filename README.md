# Waky
Waky is a web application to manage Wake-On-Lan supporting devices

## Requirements
* Python 3
* Bower
* tox

# Local dev
```
docker run -it --rm -v $PWD:/workspace -w /workspace ubuntu /bin/bash
tox -e dev
WAKY_CONF=waky.conf
env/bin/waky
```

# Testing
```
tox
```

## Run
```
tox -e run
```

## Release
```
bump2version minor
```

## Build
```
tox -e build
```

## Build & Publish to PyPI
```
tox -e upload
```
